# 🌤️ Weather Dashboard

A beautiful real-time weather dashboard built with HTML, CSS and JavaScript using the OpenWeatherMap API.

## ✨ Features

- Search weather by any city
- Shows temperature, humidity, wind speed
- Shows visibility, pressure, sunrise & sunset
- Beautiful dark UI with animations
- Works on mobile and desktop

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- OpenWeatherMap API

## ▶️ How to Run

1. Get a free API key from [openweathermap.org](https://openweathermap.org)
2. Open `weather_dashboard.html`
3. Replace `YOUR_API_KEY` with your actual key
4. Open in browser and search any city!
